# 合规智能体页面截图说明

本文档包含合规智能体系统所有主要页面的截图，用于产品展示和文档说明。

## 截图列表

### 1. 内容录入页面
**文件**: `compliance_frontend_page1_content_input.png`

**页面功能**:
- 文本内容输入区域
- 文档导入功能（支持Word、PDF、TXT格式）
- 平台选择（抖音、小红书、微信、微博）
- 开始审核按钮

**主要特点**:
- 支持直接输入文本或导入文档
- 可选择多个发布平台进行适配审核
- 界面简洁，操作便捷

---

### 2. 内容分析页面 - 审核建议视图
**文件**: `compliance_frontend_page2_analysis_review.png`

**页面功能**:
- 审核结果展示
- 原文内容与审核建议并排显示
- 分段显示功能
- 审核建议列表（按优先级排序）
- 替换词建议
- 手动输入替代词
- 审核补充意见

**主要特点**:
- 原文中违规词红色高亮显示
- 已修改内容绿色高亮显示
- 资质审核优先显示（标记"优先"标签）
- 支持点击违规词定位到对应审核建议
- 支持分段显示和连续显示切换

---

### 3. 内容分析页面 - 对比查看视图
**文件**: `compliance_frontend_page2_analysis_comparison.png`

**页面功能**:
- 原文内容与修改后内容左右对比
- 清晰展示修改前后的差异

**主要特点**:
- 左右分栏对比显示
- 便于查看所有修改内容
- 直观展示审核效果

---

### 4. 内容分析页面 - 修改后内容视图
**文件**: `compliance_frontend_page2_analysis_modified.png`

**页面功能**:
- 仅显示修改后的合规内容
- 已修改部分绿色高亮显示
- 导出功能（复制、导出Word、导出报告）

**主要特点**:
- 专注于修改后的最终内容
- 便于复制和使用
- 支持多种导出格式

---

### 5. 知识库上传页面
**文件**: `compliance_frontend_page3_knowledge_upload.png`

**页面功能**:
- 知识库分类选择（资质审核、国家法律、平台规则、公司规则、业务规则）
- 文件上传区域（支持拖拽上传）
- 支持PDF、Word、TXT等格式
- 上传文件按钮

**主要特点**:
- 支持按分类上传知识库文件
- 拖拽上传，操作便捷
- 支持多种文档格式
- 文件大小限制提示（50MB）

---

## 页面功能总览

### 核心功能模块

1. **内容录入模块**
   - 文本输入
   - 文档导入
   - 平台选择

2. **内容分析模块**
   - 智能审核
   - 违规识别
   - 修改建议
   - 内容对比
   - 导出功能

3. **知识库管理模块**
   - 分类管理
   - 文件上传
   - 规则维护

### 审核流程

```
内容录入 → 开始审核 → 智能分析 → 审核建议 → 内容对比 → 导出结果
```

### 审核维度

1. **资质审核**（优先）
   - 医疗资质
   - 金融资质
   - 保险资质

2. **法律规则**
   - 广告法
   - 医疗广告管理办法
   - 互联网广告管理办法

3. **平台规则**
   - 抖音
   - 小红书
   - 微信
   - 微博

4. **公司规则**
   - 内容审核规范
   - 宣传规范

5. **业务规则**
   - 广告用语规范
   - 价格宣传规范
