# 🎯 智策营销智能体平台 - 全新重构开发计划

## 项目概述

基于"视频混剪智能体前端开发方案（最终合并版）"，重新设计和实现智策营销智能体平台。

## 核心设计理念

### 1. 动态布局模式
- **对话模式 (chat)**: ChatCanvas占满全屏，纯对话交互
- **创作模式 (creation)**: ChatCanvas (w-1/3) + MainCanvas (w-2/3)

### 2. 通用画布架构
- MainCanvas.vue 作为通用容器，可承载任何智能体应用
- 智能体组件独立，易于扩展

### 3. 极简导航体验
- Gemini风格的极窄侧边栏
- 仅显示图标，悬停显示Tooltip
- 减少视觉干扰，专注内容创作

## 技术栈

- **框架**: Vue 3 (Composition API, `<script setup>`)
- **构建工具**: Vite
- **状态管理**: Pinia
- **样式**: Tailwind CSS
- **UI组件**: Headless UI
- **图标**: lucide-vue-next
- **路由**: Vue Router
- **类型检查**: TypeScript

## 组件交互状态

### 主行动按钮
- **默认**: 渐变色背景
- **悬停**: `hover:brightness-110 hover:scale-105`
- **点击**: `active:scale-95`
- **禁用**: `disabled:opacity-50 disabled:cursor-not-allowed`

### 输入框
- **默认**: bg-secondary + border-color
- **聚焦**: `focus:border-blue-500 focus:ring-2 focus:ring-blue-500/50`

### 图标按钮
- **默认**: text-secondary
- **悬停**: text-primary + bg-secondary背景
