# 合规知识库文件示例

本文档提供合规知识库文件的格式示例，供上传到 Coze 知识库时参考。

## 一、资质审核规则文件示例

### 1.1 医疗资质审核规则.md

```markdown
# 医疗资质审核规则

## 规则说明
我公司（天九网科）无医疗相关资质/牌照，因此不能提及医疗相关内容。涉及健康相关内容需使用"健康"、"保健"等替代。

## 违规关键词列表
- 医疗
- 治疗
- 诊断
- 处方
- 疗效
- 治愈
- 医疗资源
- 医院
- 医生
- 药品
- 医疗器械
- 医疗服务
- 疾病
- 病症
- 症状
- 康复
- 痊愈

## 替换词建议
| 违规词 | 替换词1 | 替换词2 | 替换词3 |
|--------|---------|---------|---------|
| 医疗 | 健康 | 保健 | - |
| 治疗 | 支持 | 辅助 | 帮助 |
| 诊断 | 建议 | 评估 | 分析 |
| 处方 | 方案 | 计划 | - |
| 疗效 | 效果 | 作用 | - |
| 治愈 | 改善 | 提升 | - |
| 医疗资源 | 健康支持 | 保健服务 | - |
| 医疗服务 | 健康服务 | 保健服务 | - |

## 审核案例

### 案例1
**原文：** 我们提供专业的医疗服务
**修改：** 我们提供专业的健康服务
**原因：** 公司无医疗资质，不能使用"医疗"一词

### 案例2
**原文：** 我们的医疗团队为您提供治疗方案
**修改：** 我们的健康团队为您提供支持方案
**原因：** 公司无医疗资质，不能使用"医疗"、"治疗"等词

### 案例3
**原文：** 通过我们的医疗资源，帮助客户快速康复
**修改：** 通过我们的健康支持，帮助客户快速改善
**原因：** 公司无医疗资质，不能使用"医疗资源"、"康复"等词

## 优先级
critical（最高优先级）

## 规则类型
qualification（资质审核）
```

## 二、内容审核规则文件示例

### 2.1 绝对化用语禁止规则.md

```markdown
# 绝对化用语禁止规则

## 规则说明
根据《广告法》规定，广告中严禁使用"最高级"、"最佳"、"最好"、"第一"、"唯一"、"绝对"等绝对化用语。

## 违规关键词列表
- 最好
- 最佳
- 第一
- 唯一
- 绝对
- 最高级
- 最专业
- 最优质
- 最权威
- 最全面
- 行业第一
- 市场第一
- 唯一选择

## 替换词建议
| 违规词 | 替换词1 | 替换词2 | 替换词3 |
|--------|---------|---------|---------|
| 最好 | 优质的 | 专业的 | 领先的 |
| 最佳 | 优秀的 | 专业的 | - |
| 第一 | 领先的 | 知名的 | 优秀的 |
| 唯一 | 专业的 | 优质的 | - |
| 绝对 | 相对 | 较为 | - |
| 最专业 | 专业的 | 优质的 | - |
| 行业第一 | 行业领先 | 行业知名 | - |

## 审核案例

### 案例1
**原文：** 选择我们，就是选择最好的服务
**修改：** 选择我们，就是选择优质的服务
**原因：** 广告法禁止使用"最好"等绝对化用语

### 案例2
**原文：** 我们是行业第一的数字化转型服务商
**修改：** 我们是行业领先的数字化转型服务商
**原因：** 广告法禁止使用"第一"等绝对化用语

### 案例3
**原文：** 这是您唯一的选择
**修改：** 这是您专业的选择
**原因：** 广告法禁止使用"唯一"等绝对化用语

## 优先级
high（高优先级）

## 规则类型
legal（法律规则）

## 相关法规
- 《广告法》第九条
- 《广告绝对化用语执法指南》
```

- 未知
- 危险
- 恐惧
- 焦虑
- 充满未知

- 未知风险

| 违规词 | 替换词1 | 替换词2 | 替换词3 |
|--------|---------|---------|---------|
| 风险 | 需要陪伴 | 共同成长 | 积极面对 |
| 未知 | 积极面对 | 共同探索 | 需要支持 |
| 危险 | 保驾护航 | 安全可靠 | - |
| 充满未知风险 | 需要我们的支持与陪伴 | 需要我们共同面对 | - |
| 充满危险 | 需要我们的保驾护航 | 安全可靠 | - |

**原文：** 面对充满危险的市场环境
**修改：** 面对需要保驾护航的市场环境
**原因：** 避免使用"危险"等引起焦虑的词

### 案例3
**原文：** 投资有风险，需谨慎
**修改：** 投资需要我们的专业支持
**原因：** 避免使用"风险"等引起焦虑的词

## 优先级
normal（普通优先级）

## 规则类型
company（公司规则）

## 相关规范
- 天九网科发〔2024〕54号 天九网科集团内容审核规范

### 2.3 内容真实性要求规则.md

```markdown
# 内容真实性要求规则

## 规则说明
根据天九网科集团内容审核管理规定，所有宣传内容必须真实、准确，不得夸大或虚假宣传。

## 检查要点
1. 不得夸大产品效果
2. 不得使用虚假数据
3. 不得做出无法实现的承诺
4. 不得使用未经证实的信息

## 违规示例

### 示例1：夸大效果
**违规：** 使用我们的产品，保证100%成功
**正确：** 使用我们的产品，有助于提升成功率

### 示例2：虚假数据
**违规：** 已有100万用户使用（实际未达到）
**正确：** 已有众多用户使用

### 示例3：无法实现的承诺
**违规：** 保证3天内见效
**正确：** 通常3-7天内可见效果

## 优先级
high（高优先级）

## 规则类型
company（公司规则）

## 相关规范
- 天九网科发〔2025〕7号 天九网科集团老板云中心内容审核管理规定
```

## 四、知识库文件组织建议

### 4.1 文件结构
```
合规知识库/
├── 01_资质审核（优先）/
│   ├── 医疗资质审核规则.md
│   ├── 金融资质审核规则.md
│   └── 保险资质审核规则.md
├── 02_国家法律/
│   ├── 广告法相关规则.md
│   ├── 医疗广告管理办法.md
│   └── 互联网广告管理办法.md
├── 03_平台规则/
│   ├── 抖音平台规则.md
│   ├── 小红书平台规则.md
│   └── 微信平台规则.md
└── 04_公司规则/
    ├── 内容审核规范.md
    ├── 避免焦虑表述规则.md
    └── 内容真实性要求规则.md
```

### 4.2 文件命名规范
- 使用中文命名，清晰描述规则内容
- 资质审核文件以"资质审核"开头
- 按优先级和类型分类组织

### 4.3 文件格式要求
- 使用 Markdown 格式
- 包含规则说明、关键词列表、替换建议、案例等
- 明确标注优先级和规则类型
